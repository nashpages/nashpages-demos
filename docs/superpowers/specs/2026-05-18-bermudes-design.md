## Sumário

8ª demo Premium do cluster Nashpages e **3ª não-médica** (LBB, Andrade GC, Bermudes). O escritório de **maior peso histórico** do cluster: fundado em 1969 pelo lendário Sérgio Bermudes (✝ 2025), 130 profissionais, 4 cidades (Rio sede + SP + BH + Brasília), Chambers Global, 19 anos consecutivos "Mais Admirado".

**Decisão estratégica do user:** site **excepcional, "que eles não vão recusar"**. NÃO replicar template LBB. Identidade visual totalmente custom (Fraunces, paleta nova), 8 sections (não 7), 3 sections inéditas no cluster (Memorial Sérgio, Caso Herzog, 4 Escritórios full-bleed com hover).

**Ângulo central:** **Legado + presente + alcance nacional**. 57 anos de história institucional + caso fundador dos direitos humanos no Brasil + cobertura federal premium.

**Headline assinatura:** *"Vamos conversar."* (Fraunces Italic 132pt no §08 Contato) — closing editorial monumental sem CTA pill startup.

## Comportamento técnico

Reusar pattern do cluster onde possível, MAS com diferenças importantes:

1. **Smooth scroll baseline:** Lenis 1.3.23 duration 1.1s (mesmo do cluster)
2. **Scroll híbrido Hero↔Sobre:** capture-phase wheel listener (mesmo pattern LBB) — REAPROVEITAR a lógica do LBBSmoothScroll/AndradeGCSmoothScroll com renaming
3. **Nav cor adaptativa:** branco no Hero dark, tinta no resto. CTA "FALAR" reveal pós-Hero (mesmo pattern)
4. **§06 Hover animation** ⭐ NOVO — implementar:
   - Overlay opacity: `0.72 → 0.45` em 400ms ease-out
   - Foto: `scale(1) → scale(1.05)` em 600ms ease-out (parent overflow-hidden)
   - Arrow: `translateX(0) → translateX(8px)` em 300ms ease-out + opacity `0.5 → 1`
5. **Reduced-motion:** respeitar `prefers-reduced-motion`
6. **Marquee Footer:** "BERMUDES · BERMUDES · ..." opacity 0.04 estático
